# comrade
Helps access remote shell through a browser

## Setup
1. Make virtualenv
2. Activate virtualenv
3. `cd path/to/comrade`
4. `pip install -r requirements.txt`
5. `mkdir logs`


### Usage
1. `cd path/to/comrade`
2. Activate virtualenv
3. `python privet.py`
4. Open `localhost:8000` in browser.
5. Command away!
